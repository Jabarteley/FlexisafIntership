Understanding Node.js and npm

By Abduljabar

What is Node.js?

Node.js is a JavaScript runtime built on Chrome's V8 engine that lets you run JavaScript outside the browser. It’s commonly used to build backend servers and APIs.

What is npm or yarn?

npm (Node Package Manager) and Yarn are tools used to install, manage, and share JavaScript packages and dependencies for Node.js projects.

Basic npm Commands

  • npm init -y — Initialize a new Node.js project
  • npm install express — Install a package
  • npm start — Run your app (if defined in package.json)

Practice Example

const express = require('express');
const app = express();

app.get('/', (req, res) => res.send('Hello Node.js!'));
app.listen(3000, () => console.log('Server running on port 3000'));
